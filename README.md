## Argomenti principali
Repository locale / remota
Github e Gitlab

Inizializzazione repo: `git init`  \ `git clone`   

staging: `git add .`  
commit: `git commit -m "messaggio"`  
push modifiche in remoto: `git push`   

update repo: `git pull`   

Merge request / Pull request
